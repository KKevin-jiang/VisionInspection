## 一、数据流转图

```mermaid
flowchart TB
    subgraph 生产线["🏭 生产线（触发层）"]
        PLC["三菱 PLC<br/>（节拍控制）"]
    end

    subgraph 相机层["📷 相机层"]
        Cam["海康工业相机<br/>MVS SDK"]
    end

    subgraph 上位机层["🖥️ 上位机层"]
        subgraph 换型["🔀 换型模块"]
            ManualSelect["手动模式<br/>操作员下拉选型"]
            HTTP_API["HTTP API 客户端<br/>crankshaft_api.py<br/>POST /api/v1/crankshaft/model-by-serial"]
            SQL_Query["SQL Server 查询<br/>database.py<br/>(仅写结果，不再查机型)"]
        end

        subgraph 检测引擎["🔍 检测引擎 detector.py"]
            ROI_Crop["ROI 区域裁剪"]
            Binarization["传统二值化<br/>Otsu / Fixed / Range<br/>→ 形态学开运算<br/>→ 连通域面积判定"]
            AI["AI 推理 YOLOv8<br/>加载 .pt 模型<br/>→ 类别+置信度判断"]
        end

        subgraph 结果处理["📊 结果处理"]
            Judge{"合格?"}
            IO_Output["Line1 输出控制<br/>camera_manager.py"]
            DB_Write["写检测结果<br/>T_VisionResult 表"]
            Storage["本地存储<br/>storage.py"]
        end

        subgraph 存储子层["💾 存储子层"]
            ImageDir["图片<br/>{image_root}/YYYYMMDD/MachineType/SerialNo_PASS-FAIL_HHmmss.jpg"]
            CSV["CSV 日志<br/>timestamp, serial_no, machine_type, result, score..."]
            Excel["Excel 报表<br/>按日期汇总 + 合格率图表"]
        end
    end

    subgraph 外部服务["🌐 外部服务"]
        APIServer["曲轴工位 HTTP 服务<br/>192.168.0.101:8080"]
        SQLServer["SQL Server<br/>192.168.0.101\\SQL201<br/>数据库: BLT"]
    end

    subgraph UI["🖥️ UI 展示 PyQt5"]
        LiveView["实时画面<br/>（叠加 ROI 框+标注）"]
        ResultPanel["结果面板<br/>✅合格 / ❌不合格"]
        StatsPanel["统计卡片<br/>总计/合格/不合格/合格率"]
        StatusBar["状态栏<br/>相机●/数据库●/上次检测时间"]
    end

    %% === 主要数据流 ===

    PLC -->|"Line0 上升沿触发<br/>（硬件IO信号）"| Cam
    Cam -->|"采集图像<br/>（USB3/GigE）"| ROI_Crop

    ROI_Crop --> Binarization
    ROI_Crop --> AI
    Binarization --> Judge
    AI --> Judge

    %% 换型流（自动模式）
    Cam -->|"触发后"| HTTP_API
    HTTP_API -->|"POST 流水号<br/>requestId: UUID"| APIServer
    APIServer -->|"返回完整机型号<br/>如 10V3AABB1234"| HTTP_API
    HTTP_API -->|"取前4位 → 10V3<br/>加载对应配方 JSON"| ROI_Crop

    %% 换型流（手动模式）
    ManualSelect -->|"选择机型 → 锁定配方"| ROI_Crop

    %% 结果输出
    Judge -->|"合格: Line1 高电平 500ms<br/>不合格: Line1 保持低电平"| IO_Output
    IO_Output -->|"Line1 输出信号"| PLC
    Judge --> DB_Write
    DB_Write -->|"INSERT 检测结果"| SQLServer
    Judge --> Storage
    Storage --> ImageDir
    Storage --> CSV
    Storage --> Excel

    %% UI 展示流
    Cam -->|"实时帧"| LiveView
    Binarization -->|"二值化可视化"| LiveView
    AI -->|"检测框标注"| LiveView
    Judge -->|"结果"| ResultPanel
    DB_Write -->|"统计数据"| StatsPanel
    Judge -->|"状态信息"| StatusBar

    %% 样式
    classDef plc fill:#e57373,color:#fff
    classDef camera fill:#ffb74d,color:#000
    classDef core fill:#64b5f6,color:#000
    classDef external fill:#81c784,color:#000
    classDef storage fill:#ba68c8,color:#fff
    classDef ui fill:#4dd0e1,color:#000

    class PLC plc
    class Cam camera
    class HTTP_API,ROI_Crop,Binarization,AI,Judge,IO_Output,DB_Write,Storage core
    class APIServer,SQLServer external
    class ImageDir,CSV,Excel storage
    class LiveView,ResultPanel,StatsPanel,StatusBar ui
```

### 数据流转关键说明

| 步骤 | 数据 | 流向 | 说明 |
|------|------|------|------|
| ① 触发 | 硬件 IO 信号 | PLC → 相机（Line0） | 上升沿触发，50µs 防抖 |
| ② 采图 | 图像帧 | 相机 → PC（SDK） | USB3.0 / GigE Vision |
| ③ 换型（自动） | 流水号 JSON | PC → HTTP API → 配方文件 | POST 请求，取前4位机型号匹配配方 |
| ③' 换型（手动） | 机型选择 | 操作员 → UI 下拉框 | 锁定配方，忽略上位机 |
| ④ 检测 | 图像 → 结果 | detector.py | 二值化或 AI，按配方配置执行 |
| ⑤ IO 输出 | 高低电平 | PC → 相机 Line1 → PLC | 合格=高电平500ms，不合格=低电平 |
| ⑥ 写库 | 检测结果 | PC → SQL Server | INSERT T_VisionResult |
| ⑦ 存储 | 图片/CSV/Excel | PC → 本地磁盘 | 按日期+机型目录组织 |

### 实物连接清单

| 序号 | 从 | 到 | 介质/协议 | 说明 |
|------|----|----|-----------|------|
| ① | PLC Line0 输出 | 相机 GPIO 输入 | 屏蔽信号线（2芯） | 触发信号，上升沿，50µs 防抖 |
| ② | 相机 GPIO 输出 | PLC Line1 输入 | 屏蔽信号线（2芯） | 结果信号，高=合格(500ms)，低=不合格 |
| ③ | 相机数据口 | 工控机 | USB3.0 / GigE 网线 | 图像数据传输 + SDK 控制 |
| ④ | 工控机网口 | 交换机 | RJ45 网线 (Cat5e+) | TCP/IP 局域网 |
| ⑤ | 交换机 | 服务器 | RJ45 网线 (Cat5e+) | SQL Server (1433) + HTTP API (8080) |
| ⑥ | 工控机显卡 | 显示器 | HDMI / DP | 操作员界面显示 |
| ⑦ | 键鼠 | 工控机 | USB | 操作输入 |
| ⑧ | 发动机 | 定位夹具 | 机械 | 保证检测位置一致 |
| ⑨ | 相机 | 夹具上方 | 支架固定 | 拍摄 ROI 区域 |
| ⑩ | 环形光源 | 发动机 | 光学 | 均匀补光，减少阴影 |
